3.HTML属性

在编写html元素时，我们可以对着这些元素添加一些属性

例如我们在编写链接元素时，href就是给这个元素添加了一个属性

还有在编写图片元素，他的宽度和高度也是属性

这里列举部分属性

属性 描述
class 为html元素定义一个或多个类名（classname）(类名从样式文件引入)
id 定义元素的唯一id
style 规定元素的行内样式（inline style）
title 描述了元素的额外信息 (作为工具条使用)

详细各种属性可查看HTML 标准属性参考手册
